## Executando

Para rodar o app (depois de ter rodado a instalação das depedências pelo `yarn`):
Execute:
`react-native start`

Executando no Android
`react-native run-android`

Executando no iOS
`react-native run-ios`

### Executando testes

Apos execute:
`yarn test`
